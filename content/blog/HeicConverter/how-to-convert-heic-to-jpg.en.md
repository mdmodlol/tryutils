## Method 2: iPhone Settings Modification (Preventive Method)

If you want your iPhone to directly capture photos in JPG format:

1. Open iPhone **Settings**
2. Find **Camera**
3. Tap **Formats**
4. Select **Most Compatible** instead of "High Efficiency"

**Note:** This setting will cause photos to take up more storage space.

---

## Method 3: Windows 10/11 Built-in Feature

Windows 10 version 1903 and later support HEIC format:

1. Search and install **HEIF Image Extensions** in Microsoft Store
2. After installation, you can directly open HEIC files in **Paint**
3. After opening, select **File → Save As → JPEG**

---

## Method 4: Using macOS Preview App

If you're using a Mac:

1. Double-click HEIC file to open with **Preview**
2. Select **File → Export**
3. Choose **JPEG** from the format dropdown menu
4. Adjust the quality slider
5. Click **Save**
